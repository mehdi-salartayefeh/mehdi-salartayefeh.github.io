jsPage v0.1 Demo

jspage a tiny library for easy page navigation in a SPA (Single Page Application) In one page we can have several links to subpages and a div want to show all links into that. This target element must have an attribute jspage-target Links are any button or hyperlink or other elements containing jspage attribute. This attribute has the subpage url. See below example:
Page 1 Page 2 Page 3
by clicking on each above links, it's url will be loaded into this element using "JQuery.load(url)" function.
<nav> <a jspage="/subpages/page1.html">Page 1</a> <a jspage="/subpages/page2.html">Page 2</a> <a jspage="/subpages/page3.html">Page 3</a> </nav> <div jspage-target> by clicking on each above links, it's url will be loaded into this element using "JQuery.load(url)" function. </div>
